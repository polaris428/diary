# 2D 갤러리 레이아웃 디테일 최적화 (가로 스크롤)

## 작업 일시
- 날짜: 2026년 5월 14일
- 시간: 23시 54분

## 작업 내용 요약
- 가로 스크롤 전환 후 발생한 UX 문제(하단 네비게이션 간섭) 및 심미성 문제(단조로운 격자)를 해결하기 위한 레이아웃 재계산 작업 수행.

## 상세 변경 사항

### 1. 하단 네비게이션 가림 문제 해결 (`useGalleryLayout.ts`)
- **기준선(Baseline) 상향 조정**: 하단에 고정된 앱 네비게이션 바(약 88px)와 액자가 겹치는 문제를 해결하기 위해, 두 번째 행의 기준 `top` 좌표를 `58vh`에서 `65vh`로 변경하되, 기준점을 상단이 아닌 중앙으로 변경.
- **수직 중앙 정렬(Vertical Centering) 로직 적용**: 액자 높이에 따라 아래로만 길어져 하단을 침범하던 방식을 버리고, `top: calc(baseTop - halfHeight)` 방식을 도입하여 위아래로 균등하게 배치되도록 계산식 변경. (Hover Transform 시 발생하는 충돌 방지를 위해 CSS `translateY` 대신 JS로 정확한 픽셀 단위 계산)

### 2. 미술관 리듬감 부여 (액자 크기 다양화)
- **크기 혼합 배치**: 모든 액자가 똑같은 크기를 갖던 '단조로운 격자'에서 벗어나, 3가지 사이즈(S, M, L)가 특정 패턴으로 섞여서 렌더링되도록 구현.
  - S: `12vw` (가장 작음)
  - M: `16vw` (중간)
  - L: `19vw` (가장 큼)
- 이를 통해 반듯한 격자(Grid) 구조의 정돈됨을 유지하면서도, 실제 전시회에 온 것 같은 다채로운 리듬감을 확보.

### 3. 벽면 깊이감 추가 (`TheGallery.vue`)
- **조명 효과**: 너무 평면적이던 순백색 배경 상단에 아주 옅은 `linear-gradient` 조명 효과를 주어, 위에서 아래로 빛이 은은하게 떨어지는 '화이트 큐브' 갤러리 특유의 입체감을 연출.
