## 기능 06 — 일기 단건 삭제

### 개요

| 항목 | 내용 |
| --- | --- |
| 화면 | `pages/diary/[id].vue` |
| Store | `diary.store.deleteEntry(id)` |
| Supabase | `entries DELETE` |
| 성공 후 | `/diary` 이동 |

### 화면 구성

| 컴포넌트 | Props / 역할 |
| --- | --- |
| BaseButton danger | 삭제 버튼 |
| BaseDialog | `variant=danger`, `title="정말 삭제할까요?"` |

### 플로우

- 삭제 버튼 클릭
- BaseDialog 노출
- 취소 또는 확인
- 확인 시 `deleteEntry(id)`
- 성공: `/diary` 이동
- 실패: 다이얼로그 유지

### 수용 기준 (AC)

- [ ]  확인 없이 DELETE가 실행되지 않는다
- [ ]  삭제 후 목록에서 해당 항목이 사라진다
- [ ]  삭제된 id 접근 시 404 처리된다
