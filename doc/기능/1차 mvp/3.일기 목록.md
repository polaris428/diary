## 기능 03 — 일기 목록

### 개요

| 항목 | 내용 |
| --- | --- |
| 화면 | `pages/diary/index.vue` |
| Store | `entries / fetchEntries() / prevMonth() / nextMonth()` |
| Supabase | 월 범위 필터 + DESC 정렬 SELECT |

### 유저 스토리

> 나는 내 일기를 월별로 모아보고 원하는 기록에 바로 접근하고 싶다.
> 

### 화면 구성

| 컴포넌트 | Props / 역할 |
| --- | --- |
| AppHeader | `title="나의 기록"` |
| DiaryMonthNav | `year / month / disableNext` |
| DiaryCard 리스트 | `v-for store.entries` |
| BaseEmptyState | 빈 달일 때 노출 |

### 플로우

- `/diary` 진입 → `fetchEntries()`
- 목록 렌더링 또는 BaseEmptyState 표시
- 월 이동 → 내부에서 `fetchEntries()` 재호출
- 카드 클릭 → `/diary/:id`

### 검증 규칙

| 규칙 | 처리 |
| --- | --- |
| RLS | 본인 데이터만 반환 |
| 미래 월 이동 | 현재 월 초과 차단 |
| 빈 달 | BaseEmptyState 표시 |
| fetch 실패 | BaseToast error + `store.error` 저장 |

### 수용 기준 (AC)

- [ ]  목록이 `created_at DESC` 로 정렬된다
- [ ]  타인 데이터가 반환되지 않는다
- [ ]  빈 달 진입 시 BaseEmptyState가 노출된다
- [ ]  월 이동 시 해당 월 데이터로 갱신된다
- [ ]  현재 월에서 다음 달 버튼이 비활성화된다
