# AI 문서

이 디렉터리는 프로젝트에서 사용하는 AI 관련 문서를 관리한다.

## 구조

- `prompts/`: AI 에이전트별 프롬프트 문서

## 문서 목록

- `prompts/base-agent.md`: 기본 개발 작업용 AI 에이전트 프롬프트
- `prompts/code-review-agent.md`: 코드 리뷰용 AI 에이전트 프롬프트
- `prompts/logging-rules.md`: AI 작업 및 리뷰 로그 공통 작성 규칙

## 규칙

- 새 AI 문서는 목적별로 `prompts/` 아래에 추가한다.
- 문서 파일은 `*.md` 확장자를 사용한다.
- 파일명은 역할이 드러나도록 일관되게 작성한다.
