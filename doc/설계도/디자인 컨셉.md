# 디자인 컨셉: The Frame (나만의 몰입형 기록 전시관)

> "내가 직접 큐레이팅한 2D 그림 벽(Gallery Wall) 속으로 다이브하여, 그 이면의 치열한 기록(일기)을 만나는 몰입형 뮤지엄."

---

## 1. 서비스 철학: "나의 기록을 예술로 격상시키다"

- **본질**: 단순한 일기 나열이 아닌, 나의 치열한 과정과 성취를 한 점의 '마스터피스'로 대우하는 공간입니다.
- **핵심 가치**: "기록은 과정이고, 그 과정의 끝은 예술이다."
- **심리적 보상**: 내가 직접 큐레이팅한 갤러리를 통해 자기만족을 극대화하고, 신뢰 기반의 링크 공유로 나의 가치를 증명합니다.

---

## 2. 디자인 컨셉: "프리미엄 2D 갤러리 월 (Gallery Wall)"

### 🖼️ 공간 (Modern White Cube)
- **분위기**: 어두침침함을 걷어낸 밝고 화사한 **화이트 톤**의 현대 미술관 벽면.
- **공간감**: CSS 레이어링과 정교한 그림자(Box-shadow)를 통해 구현된 2D 평면 위의 입체감.
- **조명 효과**: 실제 빛 대신, 그림자의 깊이와 방향으로 공간의 은은함을 표현.

### 🎨 배치 (Salon Hang & Relative Coordinate Model)
- **스타일**: 다양한 크기와 재질(금속, 나무, 유리 등)의 액자들이 벽면에 유기적으로 모여 있는 살롱 스타일 배치.
- **상대 좌표 모델**: 향후 사용자가 액자와 오브젝트의 위치(x, y)를 자유롭게 배치할 수 있도록 캔버스 비율(%) 기반의 유연한 레이아웃 채택 (Custom Wall 확장성).
- **시각적 몰입**: 액자 겉면에는 텍스트를 일절 배제. 오직 사용자가 선택한 **'그림(썸네일)'**만 전시합니다.

### 🛠️ 커스텀 (Curator Mode) - 향후 확장
- **기능**: 사용자가 직접 액자를 드래그하여 좌표 기반으로 배치하고, 프레임 종류나 기타 오브젝트를 추가하는 '미술관 꾸미기' 환경.

---

## 3. 사용자 경험(UX): "그림 속으로의 다이브(Dive)"

1. **[전시]**: 홈 진입 시 사용자가 배치한 '그림 벽' 전체가 우아한 2D 갤러리로 나타납니다.
2. **[진입]**: 특정 그림 액자를 클릭하면 뷰포트가 해당 그림 정중앙으로 **부드럽게 확대(Scale & Translate)** 됩니다.
3. **[전이]**: 그림이 화면을 꽉 채우며 페이드아웃 효과와 함께 전시 공간이 전환됩니다.
4. **[몰입]**: 모든 군더더기가 사라진 **'완전한 백지'** 상태의 일기 화면으로 전환됩니다. 정제된 텍스트와 함께 그날의 기록을 온전히 감상합니다.

---

## 4. 핵심 기능 요약

| 기능명 | 상세 내용 |
| --- | --- |
| **작성 & 선택** | 일기 본문 작성 후, 이를 대표할 한 장의 그림(썸네일) 선택 |
| **큐레이팅** | 2D 캔버스 위에서 좌표 기반의 자유로운 액자/오브젝트 배치 (향후 확장) |
| **전시 모드** | 텍스트 없는 '그림 액자' 중심의 프리미엄 2D 갤러리 뷰 |
| **다이브(Dive)** | 클릭 시 2D 화면 확대 연출과 함께 일기 본문(텍스트 뷰)으로 전환 |

---

## ⚙️ 기술적 구현 방향 (Nuxt 3 중심)

- **Framework**: Nuxt 3
- **2D Scene**: HTML/CSS 기반의 Flexbox, Grid 또는 퍼센트(%) 좌표 시스템을 활용한 반응형 갤러리 월 구현.
- **Motion UX**: `Vue Transitions` 또는 `GSAP`을 활용해 부드러운 2D 화면 줌(Scale) 및 페이드 효과 처리.
- **State**: `Pinia`를 사용해 액자의 좌표값, 프레임 스타일, 확장 오브젝트 정보 저장.
