# 디자인 컨셉: The Frame (나만의 몰입형 기록 전시관)

> "내가 직접 큐레이팅한 그림 액자 속으로 다이브하여, 그 이면의 치열한 기록(일기)을 만나는 3D 몰입형 뮤지엄."

---

## 1. 서비스 철학: "나의 기록을 예술로 격상시키다"

- **본질**: 단순한 일기 나열이 아닌, 나의 치열한 과정과 성취를 한 점의 '마스터피스'로 대우하는 공간입니다.
- **핵심 가치**: "기록은 과정이고, 그 과정의 끝은 예술이다."
- **심리적 보상**: 내가 직접 큐레이팅한 갤러리를 통해 자기만족을 극대화하고, 신뢰 기반의 링크 공유로 나의 가치를 증명합니다.

---

## 2. 디자인 컨셉: "모던 화이트 큐브 & 살롱 행"

### 🖼️ 공간 (Modern White Cube)
- **분위기**: 어두침침함을 걷어낸 밝고 화사한 **화이트 톤**의 현대 미술관.
- **공간감**: 높은 층고, 매끄러운 바닥 반사광.
- **조명**: 작품을 돋보이게 하는 깔끔하고 전략적인 조명 설계.

### 🎨 배치 (Salon Hang)
- **스타일**: 다양한 크기와 재질(금속, 나무, 유리 등)의 액자들이 벽면에 유기적으로 모여 있는 클래식한 살롱 스타일.
- **시각적 몰입**: 액자 겉면에는 텍스트를 일절 배제. 오직 사용자가 선택한 **'그림(썸네일)'**만 배치하여 시각적 몰입감을 극대화합니다.

### 🛠️ 커스텀 (Curator Mode)
- **기능**: 사용자가 직접 액자를 드래그하여 배치하고, 프레임의 종류를 선택하는 '미술관 꾸미기' 환경을 제공합니다.

---

## 3. 사용자 경험(UX): "그림 속으로의 다이브(Dive)"

1. **[전시]**: 공유 링크를 타고 들어오면 사용자가 공들여 배치한 '그림 벽' 전체가 위용 있게 나타납니다.
2. **[진입]**: 특정 그림 액자를 클릭하면 카메라가 해당 그림 정중앙으로 **급가속하며 줌인(Zoom-in)** 합니다.
3. **[전이]**: 그림(캔버스)의 입자를 통과하는 듯한 연출과 함께 3D 공간이 페이드아웃됩니다.
4. **[몰입]**: 모든 군더더기가 사라진 **'완전한 백지'** 상태의 일기 화면으로 전환됩니다. 정제된 텍스트와 함께 그날의 기록을 온전히 감상합니다.

---

## 4. 핵심 기능 요약

| 기능명 | 상세 내용 |
| --- | --- |
| **작성 & 선택** | 일기 본문 작성 후, 이를 대표할 한 장의 그림(썸네일) 선택 |
| **큐레이팅** | 3D 벽면에서 액자 위치 드래그 배치, 프레임 스타일 변경 |
| **전시 모드** | 텍스트 없는 '그림 액자' 중심의 갤러리 뷰 |
| **다이브(Dive)** | 클릭 시 그림 속으로 빨려 들어가며 일기 본문(텍스트 뷰)으로 전환 |

---

## ⚙️ 기술적 구현 방향 (Nuxt 3 중심)

- **Framework**: Nuxt 3
- **3D Scene**: `TresJS`를 활용해 화이트 큐브 공간과 액자 오브제 렌더링.
- **Camera Motion**: `GSAP`의 ScrollTrigger나 Timeline을 사용하여 그림 속으로 빨려 들어가는 'Dive' 연출.
- **State**: `Pinia`를 사용해 사용자가 배치한 액자의 좌표값과 스타일 정보 저장.
