### 3-2. 라우트 가드 규칙

| 조건 | 처리 |
| --- | --- |
| 세션 없음 + 보호 라우트 접근 | `/` 리다이렉트 |
| 세션 있음 + `/` 접근 | `/home` 리다이렉트 |
| 공개 라우트 | `/`, `/auth/callback` |

### 3-3. 네비게이션 흐름

- 로그인 전: `/` → 구글 OAuth → `/auth/callback` → `/home`
- 로그인 후: `/home` ↔ `/diary` ↔ `/settings`
- 일기 흐름: `/home` 또는 `/diary` → `/write` → 저장 성공 → `/home`
- 상세 흐름: `/diary` → `/diary/:id` → 뒤로가기 → `/diary`
