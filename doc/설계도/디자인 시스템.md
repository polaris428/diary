# 디자인 시스템

> 토큰 정의 기준: `design-system/tokens.ts`

## 1. 원칙

- **Modern White Cube**: 밝고 정제된 현대 미술관의 분위기를 최우선으로 한다. 
- **Visual First**: 전시 모드에서는 텍스트를 배제하고 시각적 이미지(액자)에만 집중한다.
- **Dive UX**: 모든 인터랙션은 '공간으로 빨려 들어가는' 시각적 전이를 고려한다.
- 색상값 직접 하드코딩 금지 및 공통 컴포넌트는 토큰만 소비한다.
- 기능 정책 및 화면 흐름은 관련 기능 문서에서 다룬다.

## 2. 컬러 토큰 (The Frame Theme)

| 그룹 | 역할 | 특성 |
| --- | --- | --- |
| `primary` | 주요 액션·강조 | 모던한 포인트 컬러 (Deep Black / Accent) |
| `neutral` | 배경·공간·구분선 | 화이트(0), 오프화이트(100), 라이트그레이(200) 계열 |
| `gallery.wall` | 전시벽 배경 | 화이트 큐브 느낌의 정제된 배경색 |
| `gallery.floor` | 바닥 반사광 | 매끄러운 바닥면의 반사 컬러 |
| `semantic.error` | 에러 | 레드 계열 |

## 3. 액자(Frame) 토큰

사용자가 큐레이팅 모드에서 선택할 수 있는 액자 스타일입니다.

| 스타일 | 한글명 | 재질(Material) 특성 |
| --- | --- | --- |
| `Classic Gold` | 클래식 골드 | 화려한 금속 광택, 조각 장식 |
| `Modern Black` | 모던 블랙 | 심플하고 얇은 무광 프레임 |
| `Natural Wood` | 내추럴 우드 | 따뜻한 오크/월넛 나무 질감 |
| `Floating Glass` | 플로팅 글래스 | 프레임 없이 유리만 있는 연출 |

## 4. 타이포그래피 토큰

| 토큰 | 폰트 | 사용처 |
| --- | --- | --- |
| `font.serif` | Noto Serif KR | 일기 본문 (Dive 후 몰입 모드 전용) |
| `font.sans` | Pretendard | UI 전반, 시스템 가이드 |

- **몰입 모드**: 'Dive' 후 감상 화면에서는 `font.serif`를 사용하여 종이 질감의 백지 상태를 연출한다.
- 줄 간격: 기본 `1.6`, 장문 본문 `1.8`

## 5. 3D 공간 및 특수 효과

| 구분 | 토큰 | 설명 |
| --- | --- | --- |
| Lighting | `Spotlight` | 선택된 액자를 위에서 아래로 조준하는 강조광 |
| Motion | `Dive.Easing` | `GSAP Expo.inOut` (급가속 후 부드러운 안착) |
| Motion | `Dive.Duration` | 1.3s (공간감 확보를 위한 최적 시간) |
| Shadow | `ContactShadow` | 액자가 벽에 밀착되었을 때 생기는 진한 그림자 |

## 6. 공통 컴포넌트

공통 컴포넌트는 기본적으로 `shared/components`에 둔다.

### BaseButton (Visual Optimized)
- **컨셉**: 3D 공간에서는 최소한의 `ghost` 버튼 위주로 사용한다.
- **Dive 시**: 카메라 이동 중 모든 버튼은 페이드아웃 처리된다.

### BaseFrame (3D 전용)
- **역할**: 갤러리 벽면에 배치되는 3D 액자 오브제.
- **Props**: `thumbnail(이미지) / type(프레임종류) / position(좌표)`
- **동작**: 클릭 시 해당 오브제를 타겟으로 하는 Dive UX 실행.

### AppHeader / AppBottomNav
- **전시 모드**: 화이트 큐브 몰입을 위해 스크롤 시 또는 특정 상태에서 자동으로 숨겨짐.
